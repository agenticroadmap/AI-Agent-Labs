# نمونه خروجی — RAG Agent

این فایل نشان می‌دهد RAG Agent چگونه ابتدا از فایل دانش محلی استفاده می‌کند و سپس پاسخ فارسی تولید می‌کند.

---

## سؤال کاربر

```text
LangGraph برای چه پروژه‌هایی مناسب است؟
```

## Context پیدا شده از فایل دانش

```text
LangGraph برای پروژه‌هایی مناسب است که نیاز به کنترل دقیق جریان اجرا دارند؛ مثل:

- Agentهای چندمرحله‌ای
- سیستم‌های Multi-Agent
- workflowهای تحقیق و تحلیل
- Agentهایی که نیاز به memory یا state دارند
- فرآیندهایی که باید قابل مشاهده، قابل توقف و قابل ادامه باشند
```

## پاسخ Agent

```text
LangGraph برای پروژه‌هایی مناسب است که اجرای چندمرحله‌ای، کنترل دقیق جریان، memory یا state نیاز دارند. 
مثلاً برای ساخت Agentهای تحقیق، سیستم‌های Multi-Agent، workflowهای تحلیلی و فرایندهایی که باید قابل مشاهده یا قابل ادامه باشند، انتخاب مناسبی است.
```
