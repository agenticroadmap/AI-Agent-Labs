# دانش نمونه برای RAG Agent

## LangGraph چیست؟

LangGraph یک فریم‌ورک برای ساخت Agentها و workflowهای هوشمند است.  
تمرکز اصلی LangGraph روی ساخت جریان‌های stateful، قابل کنترل و چندمرحله‌ای است.

## LangGraph برای چه پروژه‌هایی مناسب است؟

LangGraph برای پروژه‌هایی مناسب است که نیاز به کنترل دقیق جریان اجرا دارند؛ مثل:

- Agentهای چندمرحله‌ای
- سیستم‌های Multi-Agent
- workflowهای تحقیق و تحلیل
- Agentهایی که نیاز به memory یا state دارند
- فرآیندهایی که باید قابل مشاهده، قابل توقف و قابل ادامه باشند

## تفاوت RAG با پاسخ معمولی مدل

در پاسخ معمولی، مدل فقط بر اساس دانش خودش پاسخ می‌دهد.  
اما در RAG، ابتدا اطلاعات مرتبط از یک منبع بیرونی مثل فایل، دیتابیس یا مستندات پیدا می‌شود و سپس پاسخ بر اساس همان اطلاعات ساخته می‌شود.

## چرا RAG برای AI Agent مهم است؟

چون Agentهای واقعی معمولاً باید با اطلاعات اختصاصی کار کنند؛ مثل:

- مستندات داخلی شرکت
- فایل‌های PDF
- دیتابیس مشتریان
- محتوای سایت
- گزارش‌های فنی
