# 01 — First Agent

اولین ایجنت ساده در AgenticRoadmap Labs.

## هدف

در این Lab یک ایجنت ساده می‌سازیم که سؤال کاربر را می‌گیرد و با کمک مدل زبانی، پاسخ فارسی تولید می‌کند.

## این Agent چه کار می‌کند؟

```text
User Question
↓
AI Model
↓
Persian Answer
```

## اجرا روی سیستم شخصی

این پروژه روی خود GitHub اجرا نمی‌شود.

برای اجرا باید ریپو را روی سیستم خود Clone کنید:

```bash
git clone https://github.com/agenticroadmap/AI-Agent-Labs.git

cd AI-Agent-Labs/labs/01-first-agent

pip install -r requirements.txt

python main.py
```
